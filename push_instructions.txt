# 推送代码到GitHub的几种方法

## 方法1：在终端中手动推送（推荐）

打开终端，执行：
cd /Users/Developer/ParentDoctorApp/Server
git push origin main

当提示输入用户名和密码时：
- 用户名：你的GitHub用户名
- 密码：使用GitHub Personal Access Token（不是账户密码）

获取Token：
1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 选择 "repo" 权限
4. 生成后复制token，用作密码

## 方法2：使用SSH（如果已配置）

cd /Users/Developer/ParentDoctorApp/Server
git remote set-url origin git@github.com:Daniel777611/ParentDoctorServer.git
git push origin main

## 方法3：使用GitHub Desktop

1. 打开GitHub Desktop
2. 打开仓库：/Users/Developer/ParentDoctorApp/Server
3. 点击 "Push origin" 按钮

## 方法4：在Xcode中推送

1. 打开Xcode
2. Source Control → Push
3. 输入GitHub凭据
